# Doxygen Note

## Comment Blocks

1. You can use either
	```
	/** Line 1
	 *  Line 2
	 *  Line 3
	 */
	Constructor()
	```
	or
	
	```
	Constructor() /**< Line 1
				   *   Line 2
				   *   Line 3
				   */

## Special Comannds

1. To jump to a function or parameter, you can use `@see FOO` to point it out.

2. To display a parameter and describe its usage, you can use `@param FOO The example parameter.`
